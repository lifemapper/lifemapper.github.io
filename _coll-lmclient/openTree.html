Lifemapper Open Tree of Life (OTL) Module

With this release, we have added a module that can be used to integrate Open Tree of Life ( http://blog.opentreeoflife.org ) services with Lifemapper services.

  • getOTLHint
  • getOTLTreeWeb

Get Open Tree of Life Hint

The "getOTLHint" function calls the Open Tree of Life hint service and returns a list of matching Open Tree of Life tree identifiers for the provided taxa name.

Example: Find the matching OTL tree ids for 'Acer' >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> treeIds = cl.otl.getOTLHint('Acer') >>> cl.close()

Get Open Tree of Life Tree

The "getOTLTreeWeb" function allows you to retrieve an Open Tree of Life tree (in Newick format) from an Open Tree of Life tree identifier. Note, this must be converted to JSON before it can be used with the Lifemapper RAD services.

Example: Get OTL tree 12345 >>> cl = LMClient(userId='someUser', pwd='somePassword') >>> otlTree = cl.otl.getOTLTreeWeb(12345) >>> cl.close()
